# PRD: Add GPLv3 LICENSE

## Goal
Add the GNU General Public License v3.0 to the repository to define the project's open-source terms.

## Requirements
1.  Add a `LICENSE` file in the root directory containing the full text of the GPLv3.
2.  Ensure the file is correctly formatted.
3.  Commit and push the change to GitHub.

## Completion Criteria
-   `LICENSE` file exists in the root.
-   File content matches GPLv3.
-   Changes are pushed to `origin/master`.
